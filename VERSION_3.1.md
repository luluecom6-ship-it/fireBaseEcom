# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-04-11
### Added
- Created `DOCUMENTATION.md` for project overview.
- Created `CHANGELOG.md` to track project history.

### Fixed
- Fixed "HTTP error! status: 0" when saving system configuration.
- Improved `robustFetch` utility to handle opaque responses (status 0).
- Updated `useSystemConfig` to use `cors` mode for better server synchronization.
- Fixed search response handling to support multiple backend data formats.

## [1.0.0] - 2026-04-09
### Added
- Initial release of the E-Com Monitoring App.
- Real-time Matrix Intelligence dashboard.
- Attendance tracking system.
- Order upload and search functionality.
- Admin panel for system configuration.
