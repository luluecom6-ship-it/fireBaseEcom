# E-Com Monitoring App Documentation

## Overview
This application is designed for monitoring e-commerce operations, specifically focusing on order picking, attendance tracking, and real-time matrix intelligence.

## Features
- **Dashboard**: Real-time overview of operations.
- **Matrix Intelligence**: Detailed ageing and store-wise distribution of orders.
- **Order Upload**: Interface for pickers to upload order completion images.
- **Attendance**: Punch-in/out system with shift tracking.
- **Admin Panel**: User management and system configuration.
- **Search**: Search for specific orders by ID.
- **Alerts**: Real-time notifications for operational issues.

## Tech Stack
- **Frontend**: React, TypeScript, Tailwind CSS, Motion (for animations).
- **Icons**: Lucide React.
- **Charts**: Recharts.
- **Data Visualization**: D3.js (where applicable).

## Project Structure
- `src/components`: Reusable UI components.
- `src/hooks`: Custom React hooks for data fetching and state management.
- `src/pages`: Main application pages.
- `src/services`: API interaction logic.
- `src/utils`: Helper functions and formatters.
